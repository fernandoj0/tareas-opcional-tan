Explicacion de algunos términos de JavaScript y HTML

function
Esta palabra reservada de JavaScript permite declarar funciones. La syntaxis es la siguiente: function(parametros) { bloque_de_coidgo }

let
Esta palabra resrvada permite declarar variables en JavaScript. A diferencia de var, el alcance de la variable es local.
const
Esta palabra resrvada permite declarar variables en JavaScript. A diferencia de var y de let, las variables con const no permiten reasignaciones de nuevos valores (a excepción de los arrays). Ni mucho menos cambios de tipo de datos.
figure
Este elemento de HTML permite incluir un texto debajo de la imagen (figcaption) para describir la imagen y mostrarlo en la página web.
label
Este elemento HTML permite colocar algun texto encima algún input y enlazarlo con éste, con la ventaja de que si el usuario cliquea dicho texto, automaticamente el cursor se coloca en el input para comenzar a recibir texto.
video
Muy parecido a img pero éste permite introducir algún vídeo dentro de la página web.
